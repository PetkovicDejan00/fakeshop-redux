# Shopnetic

React E-Commerce responsive website created using typescript, react query, redux, router, axios, react-toastify & aos packages.

Available options:
- Filtering all existing products with 4 categories & clearing the filter option.
- Selecting any of the products opens up selected product page.
- Login option, in order to see cart and use it's features, login is required.
- Option to add product/s to the cart.
- Option to choose product quantity before adding it to the cart.
- Removing one chosen product from the cart.
- Removing all products from the cart at once.
- Mobile version (website is responsive).

## Running instructions

Install the needed dependencies

```bash
  npm install
```

Once the setup is complete, run the project

```bash
  npm run dev
```
